# Dictionary Cli

## about :
- dict is dictionry on the terminal and translate word english to persian and pronunciation word english 

## run :
- `git clone https://github.com/stinerd/dictionary_cli.git`
- `pip install -r requirements.txt`
- `chmod +x install.sh`
- `./install.sh`
- `dic word`
